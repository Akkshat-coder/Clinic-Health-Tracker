# Project Statement
1.Problem Statement
In many small clinics, keeping track of patient records with pen and paper can be a real hassle. Its slow to jot down vital sign by hand, calculating things like BMI can lead to mistakes, and its tough to keep an accurate history of each patient health over time.
2. Project Objective
To build a digital "Patient Health Record System" that allows clinic staff to efficiently register patients, automate health metric calculations like BMI and maintain a persistent history of patient visits using a simple file based storage system.
3. Scope
The system focuses on:
Registration:Capturing patient details.
Calculation:Automating BMI and health categorization.
History:Storing multiple visit records for the same patient.
Persistence:Saving data to a text file so records are not lost when the program closes.
4.Target Users
Clinic Receptionists.
Nurses and Doctors.
5.High-Level Features
Patient Management:Adds new patient profiles with unique IDs.
Vitals Tracker:Input current weight and height to instantly get BMI and health status.
Search & View:Retrieve patient details by ID and view their full checkup history.
Data Persistence:Automatic saving and loading of data using file handling.
